дҪҝз”ЁMeiliSearchжҗңзҙўиҖҒиӢҸзҡ„еҚҡе®ў

еҸ‘иЎЁдәҺ 2023-10-02 жӣҙж–°дәҺ 2026-07-18 еҲҶзұ»дәҺ еҗ„з§ҚжҠҳи…ҫ Walineпјҡ йҳ…иҜ»ж¬Ўж•°пјҡ жң¬ж–Үеӯ—ж•°пјҡ 5.5k йҳ…иҜ»ж—¶й•ҝ ≈ 5 еҲҶй’ҹ
дҪҝз”Ё MeiliSearch дҪңдёәе…Ёж–ҮжЈҖзҙўеј•ж“ҺпјҢеңЁжң¬ең°жҗңзҙўиҖҒиӢҸзҡ„еҚҡе®ўгҖӮ

еҺ»е№ҙе№ҙеҲқпјҢиҖҒиӢҸжҠҳи…ҫиҝҮ MeilisearchпјҢеҪ“ж—¶йҒ—з•ҷдёӢдёҖдёӘй—®йўҳпјҢжҖҺд№Ҳз”Ё Meilisearch жқҘеҜ№иҖҒиӢҸзҡ„еҚҡе®ўиҝӣиЎҢе…Ёж–ҮжЈҖзҙўпјҢдёҚиҝҮеҫҲеҸҜжғңжІЎжңүжҲҗеҠҹ

ж–Үз« дј йҖҒй—ЁпјҡејҖжәҗе…Ёж–Үжҗңзҙўеј•ж“ҺMeiliSearch

её…е“Ҙ еҲқеҶ¬ жңҖиҝ‘еңЁйғЁзҪІе…¬еҸёејҖжәҗйЎ№зӣ®зҡ„её®еҠ©ж–ҮжЎЈпјҢиҖҒиӢҸеҸ‘зҺ°ж–ҮжЎЈжҗңзҙўеұ…з„¶йҮҮз”ЁдәҶ MeilisearchпјҢдёҖз•ӘиҜ·ж•ҷд№ӢеҗҺпјҢиҖҒиӢҸеҶіе®ҡжҚўдёӘжҖқи·ҜпјҢж—ўз„¶иҮӘе·ұдёҚдјҡж”№еҚҡж–Үзҙўеј•пјҢе°ұиҜ•иҜ• Meilisearch е®ҳж–№жҸҗдҫӣзҡ„зҲ¬иҷ« docs-scraperпјҢйҖҡиҝҮзҲ¬иҷ«жқҘз”ҹжҲҗзҡ„еҚҡж–Үзҙўеј•пјҢиҜҙе№Іе°ұе№ІгҖӮ

е®үиЈ…

еңЁзҫӨжҷ–дёҠд»Ҙ Docker ж–№ејҸе®үиЈ…гҖӮ

е®үиЈ… Meilisearch

еҰӮжһңдҪ е·Із»Ҹе®үиЈ…иҝҮдәҶ Meilisearch пјҢеҸҜд»Ҙи·іеҲ°дёӢдёҖдёӘз« иҠӮ

latest еҜ№еә”зҡ„зүҲжң¬дёә v1.3.5

  • MEILI_ENVпјҡжҢҮе®ҡдёә development жЁЎејҸпјӣ
  • MEILI_MASTER_KEYпјҡжҢҮе®ҡ master keyпјҢиҮіе°‘ 16 еӯ—иҠӮзҡ„ UTF-8 еӯ—з¬ҰдёІпјҢиҖҒиӢҸдёҖиҲ¬жҳҜз”Ё Vaultwarden зҡ„еҜҶз Ғз”ҹжҲҗеҷЁжқҘз”ҹжҲҗпјҢиҝҷжҳҜдёәдәҶдҝқжҠӨ MeilвҖӢвҖӢisearch е®һдҫӢе…ҚйҒӯжңӘз»ҸжҺҲжқғзҡ„дҪҝз”ЁпјҢжүҖд»Ҙеҝ…йЎ»еңЁеҗҜеҠЁж—¶жҸҗдҫӣдё»еҜҶй’Ҙпјӣ

еҲӣе»ә API Key

е®һдҫӢеҸ—еҲ°дҝқжҠӨеҗҺпјҢиҰҒи®ҝй—®д»»дҪ• API з«ҜзӮ№пјҢеҝ…йЎ»еҗ‘жӮЁзҡ„иҜ·жұӮж·»еҠ е…·жңүйҖӮеҪ“жқғйҷҗзҡ„е®үе…ЁеҜҶй’ҘгҖӮ

иҖҢе®һйҷ…дёҠпјҢеҪ“жӮЁз¬¬дёҖж¬ЎеҗҜеҠЁе®һдҫӢж—¶пјҢMeilisearch е·Із»ҸиҮӘеҠЁз”ҹжҲҗдәҶдёӨдёӘ API еҜҶй’ҘпјҡDefault Search API Keyе’ҢDefault Admin API KeyгҖӮ

е…¶дёӯпјҡ

  • DefaultВ SearchВ APIВ Key пјҡеҸӘиғҪз”ЁдәҺи®ҝй—®жҗңзҙўи·Ҝеҫ„пјӣ
  • DefaultВ AdminВ APIВ Key пјҡеҸҜд»Ҙи®ҝй—®йҷӨ /keys д№ӢеӨ–зҡ„жүҖжңү API и·Ҝз”ұгҖӮжӮЁеә”иҜҘйҒҝе…ҚеңЁеҸҜе…¬ејҖи®ҝй—®зҡ„д»Јз Ғдёӯе…¬ејҖй»ҳи®Өзҡ„з®ЎзҗҶеҜҶй’ҘгҖӮ

жҺҘдёӢжқҘжҲ‘д»¬йңҖиҰҒеҒҡзҡ„е°ұжҳҜиҺ·еҸ–еҲ°иҝҷдёӨдёӘ API Key

v0.24 еҸҠд»ҘдёӢзүҲжң¬дҪҝз”ЁX-MEILI-API-KEY: apiKey,В v0.25 еҸҠд»ҘдёҠзүҲжң¬дҪҝз”ЁAuthorization: Bearer apiKey

иҝ”еӣһзҡ„з»“жһңеңЁВ https://www.json.cnВ ж јејҸеҢ–д№ӢеҗҺ

еҲӣе»ә config.json

scraper е·Ҙе…·йңҖиҰҒдёҖдёӘй…ҚзҪ®ж–Үд»¶жқҘдәҶи§ЈжӮЁжғіиҰҒжҠ“еҸ–зҡ„еҶ…е®№гҖӮиҝҷжҳҜйҖҡиҝҮжҸҗдҫӣйҖүжӢ©еҷЁпјҲдҫӢеҰӮпјҢHTMLж Үи®°пјүжқҘе®һзҺ°зҡ„гҖӮ

е°ҶдёӢйқўзҡ„ж–Үд»¶дҝқеӯҳдёә config.jsonпјҢе№¶е°Ҷе…¶дёҠдј еҲ° meilisearch ж №зӣ®еҪ•

иҝҗиЎҢ docs-scraper

docs-scraper жҳҜ Meilisearch жҸҗдҫӣе№¶з»ҙжҠӨзҡ„дёҖдёӘжҠ“еҸ–е·Ҙе…·пјҢеҸҜд»ҘиҮӘеҠЁиҜ»еҸ–жӮЁзҪ‘з«ҷзҡ„еҶ…е®№пјҢе№¶е°Ҷе…¶еӯҳеӮЁеңЁ Meilisearch зҡ„зҙўеј•дёӯгҖӮ

зҺ°еңЁеҸҜд»ҘејҖе§ӢжҠ“еҸ–дәҶ

  • MEILISEARCH_HOST_URLпјҡжҢҮе®ҡ meilisearch зҡ„ url ең°еқҖпјӣ
  • MEILISEARCH_API_KEYпјҡиҖҒиӢҸз”ЁдәҶ master keyпјӣ

еңЁжҠ“еҸ–иҝҮзЁӢдёӯпјҢ CPU еҸҜиғҪдјҡйЈҷеҚҮ

еҰӮжһңзҪ‘з«ҷеҸ‘еёғдәҶж–°зҡ„ж–Үз« пјҢйңҖиҰҒйҮҚж–°жҠ“еҸ–дёҖж¬ЎпјҢиҝҷж ·иғҪдҝқжҢҒжңҖж–°зҡ„зҙўеј•

иҝҗиЎҢ

жҠ“еҸ–е®ҢжҲҗеҗҺпјҢеңЁжөҸи§ҲеҷЁдёӯиҫ“е…Ҙ http://зҫӨжҷ–IP:7700 е°ұиғҪзңӢеҲ° Meilisearch зҡ„дё»з•Ңйқў

иҫ“е…Ҙ Admin API Key еҗҺе°ұеҸҜд»ҘејҖе§ӢдҪҝз”ЁдәҶ

жҗңзҙў admin@shinobi.videoпјҢеҫҲеҝ«е°ұиҝ”еӣһдәҶз»“жһңпјҢзӮ№й“ҫжҺҘеҸҜд»ҘзӣҙжҺҘжү“ејҖж–Үз« 

жҗңзҙўжқЎ

еңЁеҸ°ејҸжңәдёҠж–°е»әдёҖдёӘ search-bar.html ж–Үд»¶пјҢеҶ…е®№еҰӮдёӢпјҡ

  • hostUrlпјҡ жҳҜ Meilisearch зҡ„и®ҝй—®ең°еқҖпјҢеҰӮжһңиҝңзЁӢи®ҝй—®зҡ„иҜқеҸҜд»ҘеҸҚд»ЈжҲҗеҹҹеҗҚпјӣ
  • apiKeyпјҡз”ЁеүҚйқўиҺ·еҸ–зҡ„ DefaultВ SearchВ APIВ Keyпјӣ
  • indexUidпјҡе°ұжҳҜеңЁ config.json дёӯи®ҫзҪ®зҡ„ index_uid зҡ„еҖјпјӣ

зӣҙжҺҘеңЁжөҸи§ҲеҷЁдёӯжү“ејҖ search-bar.html ж–Үд»¶пјҢеҸӘдјҡеңЁе·ҰдёҠи§’зңӢдёӘдёҖдёӘж–Үжң¬жЎҶ

з»§з»ӯжҗңзҙў admin@shinobi.video

еҪ“з„¶пјҢеҰӮжһңдҪ зҶҹжӮү html зҡ„иҜқпјҢе®Ңе…ЁеҸҜд»ҘжҠҠдёҠйқўиҝҷж®ө html еөҢе…ҘеҲ°дҪ иҮӘе·ұзҡ„йЎөйқўдёӯ
