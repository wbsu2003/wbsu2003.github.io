TUIз•Ңйқўе®№еҷЁз®ЎзҗҶе·Ҙе…·Oxker

еҸ‘иЎЁдәҺ 2023-10-21 жӣҙж–°дәҺ 2025-03-01 еҲҶзұ»дәҺ еҗ„з§ҚжҠҳи…ҫ Walineпјҡ йҳ…иҜ»ж¬Ўж•°пјҡ жң¬ж–Үеӯ—ж•°пјҡ 1.8k йҳ…иҜ»ж—¶й•ҝ ≈ 2 еҲҶй’ҹ
Oxker жҳҜдёҖдёӘз®ҖеҚ•зҡ„ TUI е·Ҙе…·пјҢз”ЁжқҘжҹҘзңӢе’ҢжҺ§еҲ¶ docker е®№еҷЁгҖӮ

д»Җд№ҲжҳҜ Oxker пјҹ

Oxker жҳҜдёҖдёӘеҹәдәҺж–Үжң¬зҡ„з”ЁжҲ·з•ҢйқўпјҢз”ЁдәҺжҹҘзңӢ Docker е®№еҷЁзҡ„дҝЎжҒҜе’Ңз»ҹи®Ўж•°жҚ®гҖӮдёҖзӣ®дәҶз„¶пјҢOxker жҸҗдҫӣдәҶе®№еҷЁеҲ—иЎЁгҖҒе…¶еҪ“еүҚзҠ¶жҖҒгҖҒеҜ№зі»з»ҹиө„жәҗпјҲCPUгҖҒеҶ…еӯҳпјүзҡ„еҪұе“ҚгҖҒе®№еҷЁ IDгҖҒй•ңеғҸеҗҚз§°гҖҒеӨ§е°ҸзӯүгҖӮиҜҘеә”з”ЁзЁӢеәҸиҝҳжҸҗдҫӣз”ЁдәҺеҜјиҲӘеҗ„з§Қйқўжқҝе’Ңжү§иЎҢеҹәжң¬е‘Ҫд»ӨпјҲжҡӮеҒңгҖҒйҮҚж–°еҗҜеҠЁгҖҒеҒңжӯўгҖҒеҲ йҷӨпјүзҡ„жҺ§д»¶гҖӮ

д»Җд№ҲжҳҜ TUI пјҹ

TUI жҳҜжҢҮж–Үжң¬з”ЁжҲ·з•ҢйқўпјҲText-based User InterfaceпјүпјҢйҖҡиҝҮж–Үжң¬е®һзҺ°дәӨдә’зӘ—еҸЈеұ•зӨәеҶ…е®№пјҢе®ҡдҪҚе…үж Үе’Ңйј ж Үе®һзҺ°з”ЁжҲ·дәӨдә’гҖӮ

Oxker е’ҢиҖҒиӢҸд№ӢеүҚд»Ӣз»Қзҡ„ LazyDocker еұһдәҺеҗҢдёҖзұ»е·Ҙе…·

ж–Үз« дј йҖҒй—ЁпјҡLazyDockerпјҡеҹәдәҺз»Ҳз«ҜUIз®ЎзҗҶDocker

е®үиЈ…

еңЁзҫӨжҷ–дёҠд»Ҙ Docker ж–№ејҸе®үиЈ…гҖӮ

е®№еҷЁдёӢиҪҪ

еңЁжіЁеҶҢиЎЁдёӯжҗңзҙў oxker пјҢйҖүжӢ©з¬¬дёҖдёӘ mrjackwills/oxkerпјҢзүҲжң¬йҖүжӢ© latestгҖӮ

жң¬ж–ҮеҶҷдҪңж—¶пјҢ latest зүҲжң¬еҜ№еә”дёә 0.3.2пјӣ

д№ҹеҸҜд»Ҙз”Ёе‘Ҫд»ӨиЎҢдёӢиҪҪ

еҰӮжһңдёӢдёҚеҠЁпјҢеҸҜд»ҘиҜ•иҜ•йҖҡиҝҮ dockerproxy дёӢиҪҪпјҢдјҡеӨҡеҮәдёӨжӯҘ

еңЁ жҳ еғҸ дёӯиғҪзңӢеҲ°дёӢиҪҪзҡ„й•ңеғҸпјҢеӣ дёәжҳҜйҮҮз”Ё Rust ејҖеҸ‘зҡ„пјҢжүҖд»Ҙж–Үд»¶йқһеёёе°Ҹ

д»Җд№ҲжҳҜВ /var/run/docker.sockВ ?

/var/run/docker.sockВ жҳҜВ Docker APIВ зҡ„дё»иҰҒе…ҘеҸЈпјҢз®ҖеҚ•ең°иҜҙпјҢе®ғжҳҜВ DockerВ е®ҲжҠӨиҝӣзЁӢпјҲDocker daemonпјүй»ҳи®Өзӣ‘еҗ¬зҡ„В UnixВ еҹҹеҘ—жҺҘеӯ—пјҲUnix domain socketпјүпјҢе®№еҷЁдёӯзҡ„иҝӣзЁӢеҸҜд»ҘйҖҡиҝҮе®ғдёҺВ DockerВ е®ҲжҠӨиҝӣзЁӢиҝӣиЎҢйҖҡдҝЎгҖӮ

еӣҫзүҮжқҘиҮӘпјҡhttps://betterprogramming.pub/about-var-run-docker-sock-3bfd276e12fd

дҪҶжҳҜзҫӨжҷ–зҡ„В DockerВ з®ЎзҗҶеҷЁе№¶дёҚж”ҜжҢҒжҢӮиҪҪВ /var/run/docker.sockВ ж–Үд»¶пјҢжүҖд»Ҙиҝҷж¬ЎжҲ‘д»¬йңҖз”Ёе‘Ҫд»ӨиЎҢжқҘе®үиЈ…

е‘Ҫд»ӨиЎҢе®үиЈ…иҝҳжҳҜжҜ”иҫғз®ҖеҚ•зҡ„пјҢз”ЁВ SSHВ е®ўжҲ·з«ҜиҝһдёҠзҫӨжҷ–пјҢжү§иЎҢдёӢйқўрҹ‘Үзҡ„е‘Ҫд»Ө

иҖҒиӢҸеёёз”Ёзҡ„В SSHВ е®ўжҲ·з«Ҝдё»иҰҒжҳҜВ FinalshellВ 

е…¶дёӯпјҡ

  • -iВ пјҡд»ҘдәӨдә’жЁЎејҸиҝҗиЎҢе®№еҷЁпјҢйҖҡеёёдёҺ -t еҗҢж—¶дҪҝз”Ёпјӣ
  • -tВ пјҡдёәе®№еҷЁйҮҚж–°еҲҶй…ҚдёҖдёӘдјӘиҫ“е…Ҙз»Ҳз«ҜпјҢйҖҡеёёдёҺ -i еҗҢж—¶дҪҝз”Ёпјӣ
  • вҖ“rm пјҡеңЁе®№еҷЁйҖҖеҮәж—¶е°ұиғҪеӨҹиҮӘеҠЁжё…зҗҶе®№еҷЁеҶ…йғЁзҡ„ж–Үд»¶зі»з»ҹпјҢ--rm йҖүйЎ№дёҚиғҪдёҺ -d еҗҢж—¶дҪҝз”Ёпјӣ
  • вҖ“pull=always пјҡе§Ӣз»Ҳе°қиҜ•жӢүеҸ–жҳ еғҸпјҢеҚідҪҝжҳ еғҸеӯҳеңЁдәҺжң¬ең°гҖӮеҰӮжһңжӢүеҸ–еӨұиҙҘпјҢеҲҷдјҡдә§з”ҹй”ҷиҜҜпјӣ
