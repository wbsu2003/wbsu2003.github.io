在VMware上跑Linux Mint

发表于 2021-01-25 分类于 各种折腾 阅读次数： ℃ 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟
Linux Mint 是一种基于 Ubuntu 的 Linux 发行版。

这几天满屏都在刷 Linux Mint，原因是两个熊孩子在爸爸的电脑上玩耍时，一顿乱按，居然因导致屏保崩溃而进入了被锁定的系统桌面，当然这个漏洞已经被修复了。

这让我想起去年底在 VMware 上安装的 Linux Mint，当时只是因为网上很多人推荐，说 Linux Mint 是适合老电脑使用的替代 Windows 的系统，所以尝试了一下。

准备工作

下载 Linux Mint

Linux Mint 有三种不同风格的版本，分别提供三种不同的桌面环境。

版本 说明
Cinnamon 最现代，最新颖，全功能的桌面
MATE 更稳定，更快的桌面
Xfce 最轻量级，最稳定

Linux Mint 最流行的版本是 Cinnamon 版。Cinnamon 主要由 Linux Mint 团队开发，专为 Linux Mint 打造的桌面环境，流畅，美观，充满新特性。

这次出现 bug 的就是 Cinnamon 版

Cinnamon 版的操作界面和 Windows 比较像，包括了菜单、工具栏以及托盘图标。

其他两种版本的详细说明请参考：https://linuxmint-installation-guide.readthedocs.io/zh_CN/latest/choose.html

最新版本的下载地址：https://linuxmint.com/download.php

安装过程

新建虚拟机，我用了 典型 模式

选中下载的 linuxmint-20-cinnamon-64bit.iso

操作系统选 Liunx，版本选 Ubuntu 64 位

改了默认的名称

磁盘大小用了默认的值，存储改为了单个文件

到这里就安装完成了

运行

启动虚拟机

点击桌面的 Install Linux Mint 开始安装

选择 中文(简体)

安装类型选 其他选项

点 新建分区表

继续

在空闲分区上右键，添加

创建 1G 的 交换空间

剩下的创建主分区，格式用的默认的 Ext4

现在安装

继续

接下来是一些个性化设置，完成之后就开始文件拷贝了，安装过程会比较久，耐心等待就好了

重启之后就可以使用了

使用

Linux Mint 属于开箱即用的系统，常用的软件都已经自带了，包括图片查看器，音乐播放器，视频播放器，办公软件，邮箱系统等等

使用上感觉也相当流畅，可以考虑给老机器上试试。
